---

## 데이터셋 개요

영국 정부 기관 HFEA가 수집한 난임 시술 기록 데이터.  
전국 클리닉이 자체적으로 입력하는 구조라 **클리닉마다 기록 기준이 달라** 일부 불일치가 존재.  
이는 데이터 한계로 인정하고 모델에 그대로 사용.

**타겟 변수:** `임신 성공 여부` (0=실패, 1=성공)  
**클래스 불균형:** 실패 190,123명(74.2%) / 성공 66,228명(25.8%)  
→ 모델 학습 시 `class_weight` 또는 `scale_pos_weight` 설정 권장

## 파생 피처 (Feature Engineering)

### 기존 파생 피처 (EDA 기반)

| 피처명 | 계산 방법 | 근거 |
|---|---|---|
| `고령_여부` | 나이 ≥ 3 (만38세↑) → 1 | 38세↑ 성공률 급감 (ASRM 가이드라인) |
| `반복_시술_여부` | 총 시술 횟수 ≥ 3 → 1 | 반복 시술자 = 난이도 높은 케이스 (Domar et al., 2010) |
| `이전_출산_경험` | 총 출산 횟수 > 0 → 1 | 출산 경험자 성공률 유의하게 높음 (Toner et al., 1991) |
| `배아_이식_효율` | 이식된 배아 수 ÷ (총 생성 배아 수 + 1) | 배아 품질·활용도 지표 (Templeton & Morris, 1998) |
| `미세주입_배아_비율` | 미세주입 생성 배아 수 ÷ (총 생성 배아 수 + 1) | ICSI 활용 비율 (Palermo et al., 1992) |
| `불임_원인_개수` | 난관질환+남성요인+배란장애+자궁내막증 합산 | 복합 원인일수록 난이도 높음 (Templeton et al., 1996) |
| `본인_난자_여부` | 난자 출처 = 본인 제공 → 1 | 난자 출처별 성공률 차이 (Kawwass et al., 2013) |
| `기증_난자_여부` | 난자 출처 = 기증 제공 → 1 | 난자 출처별 성공률 차이 |
| `난자_기증자_사용여부` | 난자 기증자 나이 ≠ 알수없음 → 1 | Disguised Missing 처리 |
| `정자_기증자_사용여부` | 정자 기증자 나이 ≠ 알수없음 → 1 | Disguised Missing 처리 |
| `배아_이식_경과일_결측여부` | 결측이면 → 1 | 결측 자체가 "이식 안 함" 신호 |

### 논문 기반 추가 파생 피처 (2차 가공)

| 피처명 | 계산 방법 | 근거 논문 | 기대효과 |
|---|---|---|---|
| `eset_blast` ⭐⭐⭐ | (단일 배아 이식=1) & (배아 이식 경과일≥5) → 1 | Kawwass et al., 2013 (JAMA) | 단일 이식 OR 2.32 × Day-5 OR 1.17. 최강 조합 |
| `eggs_x_age` ⭐⭐⭐ | 수집된 신선 난자 수 × 시술 당시 나이 | Sunkara et al., 2011 (Hum Reprod) | 나이×난자 수 교호작용. 40세↑ 15개 기준 성공률 16% vs 40세↓ 40% |
| `optimal_transfer` ⭐⭐⭐ | 이식된 배아 수 == 2 → 1 | Templeton & Morris, 1998 (NEJM) | 이식 2개가 최적 임계점. 3개↑는 다태아 위험만 증가 |
| `over_transfer` ⭐⭐ | 이식된 배아 수 ≥ 3 → 1 | Templeton & Morris, 1998 (NEJM) | 과다 이식 위험 신호 |
| `eggs_optimal` ⭐⭐ | 10 ≤ 수집된 신선 난자 수 ≤ 20 → 1 | Sunkara et al., 2011 (Hum Reprod) | 성공률 최적 구간 |
| `eggs_deficient` ⭐⭐ | 수집된 신선 난자 수 < 4 → 1 | Sunkara et al., 2011 / Toner et al., 1991 | 저반응군. FSH 프록시 |
| `eggs_excess` ⭐⭐ | 수집된 신선 난자 수 > 20 → 1 | Sunkara et al., 2011 (Hum Reprod) | 과자극 구간 — 난자 질 저하 및 OHSS 위험 |
| `low_ovarian_response` ⭐⭐ | 수집된 신선 난자 수 ≤ 3 → 1 | Toner et al., 1991 (Fertil Steril) | FSH 직접값 없어 난자 수로 난소 예비력 간접 추정 |
| `very_low_response` ⭐⭐ | 수집된 신선 난자 수 == 0 → 1 | Toner et al., 1991 (Fertil Steril) | 극저반응군. 사실상 채취 실패 |
| `elderly_donor_egg` ⭐⭐ | (나이 ≥ 3) & (기증_난자_여부=1) → 1 | Templeton et al., 1996 / Kawwass et al., 2013 | 고령(만38세↑) + 기증 난자 조합. p=0.04 유의 |
| `has_blast` ⭐ | 배아 이식 경과일 ≥ 5 → 1 | Kawwass et al., 2013 (JAMA) | 포배기(Day-5) 이식 여부 |
| `male_factor_severe` ⭐ | (불임 원인-남성 요인=1) & (시술 유형=IVF) → 1 | Palermo et al., 1992 (Lancet) | 심각한 남성 불임 + IVF 선택 조합 신호 |
| `long_infertility_proxy` ⭐ | 총 시술 횟수 ≥ 4 → 1 | Templeton et al., 1996 (Lancet) | 불임 기간 96% 결측으로 직접 사용 불가 → 시술 횟수로 대체 |
| `불임_원인_개수_v2` ⭐ | 난관질환+남성요인+배란장애+자궁내막증+남성주+여성주+부부주 합산 | Templeton et al., 1996 (Lancet) | 불임 원인 단독은 예측력 약함. 조합 개수가 더 중요 |

### 파생 피처 전체 코드 (Python)

```python
import pandas as pd
import numpy as np

df_raw = pd.read_csv("train.csv")
# ... (전처리 완료된 df 기준)

# ── 기존 파생 피처 ──────────────────────────────────────────
df['고령_여부']               = (df['시술 당시 나이'] >= 3).astype(int)
df['반복_시술_여부']           = (df['총 시술 횟수'] >= 3).astype(int)
df['이전_출산_경험']           = (df['총 출산 횟수'] > 0).astype(int)
df['배아_이식_효율']           = df['이식된 배아 수'] / (df['총 생성 배아 수'] + 1)
df['미세주입_배아_비율']        = df['미세주입에서 생성된 배아 수'] / (df['총 생성 배아 수'] + 1)
df['불임_원인_개수']           = df[['불임 원인 - 난관 질환','불임 원인 - 남성 요인',
                                   '불임 원인 - 배란 장애','불임 원인 - 자궁내막증']].sum(axis=1)
df['본인_난자_여부']           = (df_raw['난자 출처'] == '본인 제공').astype(int)
df['기증_난자_여부']           = (df_raw['난자 출처'] == '기증 제공').astype(int)
df['난자_기증자_사용여부']      = (df_raw['난자 기증자 나이'] != '알 수 없음').astype(int)
df['정자_기증자_사용여부']      = (df_raw['정자 기증자 나이'] != '알 수 없음').astype(int)
df['배아_이식_경과일_결측여부'] = df_raw['배아 이식 경과일'].isna().astype(int)

# ── 논문 기반 추가 파생 피처 ────────────────────────────────
# ⭐⭐⭐ 단일 이식 + 포배기 조합 (Kawwass 2013)
df['eset_blast']             = ((df['단일 배아 이식 여부'] == 1) &
                                 (df['배아 이식 경과일'] >= 5)).astype(int)

# ⭐⭐⭐ 나이 × 난자 수 교호작용 (Sunkara 2011)
df['eggs_x_age']             = df['수집된 신선 난자 수'] * df['시술 당시 나이']

# ⭐⭐⭐ 이식 배아 수 최적/과다 구간 (Templeton & Morris 1998)
df['optimal_transfer']       = (df['이식된 배아 수'] == 2).astype(int)
df['over_transfer']          = (df['이식된 배아 수'] >= 3).astype(int)

# ⭐⭐ 난자 수 구간 분할 (Sunkara 2011)
df['eggs_optimal']           = ((df['수집된 신선 난자 수'] >= 10) &
                                 (df['수집된 신선 난자 수'] <= 20)).astype(int)
df['eggs_deficient']         = (df['수집된 신선 난자 수'] < 4).astype(int)
df['eggs_excess']            = (df['수집된 신선 난자 수'] > 20).astype(int)

# ⭐⭐ 난소 저반응군 (Toner 1991 — FSH 프록시)
df['low_ovarian_response']   = (df['수집된 신선 난자 수'] <= 3).astype(int)
df['very_low_response']      = (df['수집된 신선 난자 수'] == 0).astype(int)

# ⭐⭐ 고령 + 기증 난자 교호작용 (Templeton 1996, Kawwass 2013)
df['elderly_donor_egg']      = ((df['시술 당시 나이'] >= 3) &
                                 (df['기증_난자_여부'] == 1)).astype(int)

# ⭐ 포배기 이식 여부 단독 (Kawwass 2013)
df['has_blast']              = (df['배아 이식 경과일'] >= 5).astype(int)

# ⭐ 장기 불임 프록시 (Templeton 1996)
df['long_infertility_proxy'] = (df['총 시술 횟수'] >= 4).astype(int)

# ⭐ 불임 원인 개수 v2 — 주/부 포함 확장 (Templeton 1996)
cause_cols_v2 = ['불임 원인 - 난관 질환','불임 원인 - 남성 요인',
                 '불임 원인 - 배란 장애','불임 원인 - 자궁내막증',
                 '남성 주 불임 원인','여성 주 불임 원인','부부 주 불임 원인']
df['불임_원인_개수_v2']       = df[cause_cols_v2].sum(axis=1)

# ⭐ 심각한 남성 불임 신호 (Palermo 1992)
df['male_factor_severe']     = ((df['불임 원인 - 남성 요인'] == 1) &
                                 (df['시술 유형'] == 0)).astype(int)  # IVF Label
```

## 데이터 한계 (논문 작성 시 명시 권장)

1. **HFEA 클리닉별 기록 불일치:** 불명확 불임 원인=1이면서 다른 원인도 1인 케이스 10.8% 존재
2. **불임 원인 컬럼 구조적 모순:** 불임 원인-@(검사 기반)과 남/여/부부 주/부(의사 판단)의 기록 기준 불일치로 91,727명이 남성 요인=1이지만 남성 주/부 모두 0
3. **배아 생성 주요 이유 편향:** 저장용/기증용(8.6%)은 이식 자체를 안 해 임신 성공이 불가능한 케이스
4. **클래스 불균형:** 성공(25.8%) vs 실패(74.2%) → `scale_pos_weight` 또는 `class_weight` 설정 필요
5. **핵심 변수 부재:** AMH, FSH, LH, PRL, E2, BMI, 자궁내막 두께 등 호르몬 수치 없음 → 난자 수·나이 등으로 간접 대체 (Hassan et al., 2023; Biomedicines 2025)
6. **불임 기간 결측:** 임신 시도 경과 연수 96.3% 결측으로 직접 사용 불가. 총 시술 횟수로 간접 대체 (Templeton et al., 1996)
