## 2. 데이터에서 발견한 핵심 신호 (EDA 결과)

**이 표를 보면 어디서 점수를 올릴 수 있는지 보임.**

### 배아 이식 stage — 가장 큰 단일 신호
| 배아 이식 경과일 | 임신 성공률 | 건수 |
|---|---|---|
| 0일 (이식 없음) | **0.1%** | 24,904 |
| 2일 (cleavage) | 21.2% | 35,078 |
| 3일 (cleavage) | 25.9% | 57,924 |
| **5일 (blastocyst)** | **40.4%** | 81,459 |
| 6일 | 30.0% | 2,773 |

→ 5일 배아(blastocyst) = 가장 높은 성공률. **`blast_stage`** 파생변수가 중요.

### 특정 시술 유형 — "첫 주기 vs 반복 주기" 신호
| 유형 | 성공률 |
|---|---|
| **ICSI / BLASTOCYST** | **35.7%** |
| **IVF / BLASTOCYST** | **36.6%** |
| ICSI (일반) | 27.3% |
| IVF (일반) | 26.1% |
| **ICSI:ICSI** (반복 주기) | **1.0%** ← 매우 낮음 |
| **IVF:IVF** (반복 주기) | **1.0%** ← 매우 낮음 |

→ `X:Y` 형태 이름은 **2차 이상 반복 주기** 케이스로 보임 (성공률 급격히 낮음).  
→ 이 패턴을 추출하는 파생변수 (`is_repeat_cycle`) 생성 필요.

### 나이 × 성공률
| 나이 구간 | 성공률 |
|---|---|
| 만18-34세 | 32.3% |
| 만35-37세 | 27.8% |
| 만38-39세 | 21.7% |
| 만40-42세 | 15.9% |
| 만43-44세 | 11.8% |
| 알 수 없음 | **0.0%** ← 의심 데이터, 처리 필요 |

→ 나이는 ordinal 순서 그대로 보존이 핵심.

### 단일 배아 이식 여부 — 큰 차이
| | 성공률 |
|---|---|
| 단일 이식(eSET) | **36.7%** |
| 다중 이식 | 22.9% |

→ 단일 이식 + blastocyst 조합 = **최고 성공 조건**.

## 3. 점수 올릴 수 있는 실험 목록 (우선순위 순)

### ★★★ 즉시 효과 기대 (파생변수 추가)
```
1. blast_stage = (배아 이식 경과일 >= 5)                    ← AUC +0.01~0.02 예상
2. no_transfer = (이식된 배아 수 == 0)                      ← 성공률 0.1%, 분리 필수
3. is_repeat_cycle = 특정 시술 유형에 ':' 포함              ← 성공률 1% vs 27%+
4. eset_blast = (단일 배아 이식 == 1) AND (blast_stage == 1)← 최우수 케이스 flag
5. age_unknown_flag = (시술 당시 나이 == '알 수 없음')       ← 성공률 0%, 별도 처리
```

### ★★☆ 중간 효과 (인코딩 개선)
```
6. 특정 시술 유형 Target Encoding (fold-safe)             ← 고카디널리티 범주, 현재 단순 category
7. 시술 시기 코드 Target/Frequency Encoding              ← 의미 모르는 코드 → frequency로 노출
8. 횟수형 컬럼 "0회~6회 이상" → is_first_time flag 추가   ← 초회 vs 반복 구분
```

### ★☆☆ 모델 다각화 (나중에)
```
9.  XGBoost 추가 후 OOF 블렌딩
10. CatBoost 추가 (고카디널리티 categorical에 강함)
11. Optuna 하이퍼파라미터 튜닝 (LGBM num_leaves, min_data_in_leaf)
12. Isotonic calibration (확률 분포 보정)
```
