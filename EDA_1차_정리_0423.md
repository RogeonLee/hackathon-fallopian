## 데이터셋 개요

영국 정부 기관 HFEA가 수집한 난임 시술 기록 데이터예요.  
전국 클리닉이 자체적으로 입력하는 구조라 **클리닉마다 기록 기준이 달라** 일부 불일치가 존재해요.  
이는 데이터 한계로 인정하고 모델에 그대로 사용해요.

**타겟 변수:** `임신 성공 여부` (0=실패, 1=성공)  
**클래스 불균형:** 실패 190,123명(74.2%) / 성공 66,228명(25.8%)  
→ 모델 학습 시 `class_weight` 또는 `scale_pos_weight` 설정 권장

### ✅ 시술 당시 나이

### ✅ 시술 유형
- **처리:** Label Encoding (IVF / DI)
- **IVF (체외수정):** 난자를 체외에서 정자와 수정 후 배아를 자궁에 이식. 가장 고도의 기술
- **DI (공여 정자 인공수정):** 기증 정자를 배란기에 맞춰 자궁에 주입. 수정은 몸 안에서 발생
- **성공률:** IVF 26.2% / DI 12.9% — 데이터의 97.5%가 IVF

### ✅ 특정 시술 유형

## 파생 피처 (Feature Engineering)

| 피처명 | 계산 방법 | 근거 |
|---|---|---|
| `고령_여부` | 나이 ≥ 3 (만38세↑) → 1 | 38세 이상 성공률 급감 (ASRM 가이드라인) |
| `반복_시술_여부` | 총 시술 횟수 ≥ 3 → 1 | 반복 시술자 = 난이도 높은 케이스 (Domar et al., 2012) |
| `이전_출산_경험` | 총 출산 횟수 > 0 → 1 | 출산 경험자 성공률 유의하게 높음 (Toner et al., 1991) |
| `배아_이식_효율` | 이식된 배아 수 ÷ (총 생성 배아 수 + 1) | 배아 품질·활용도 지표 (Templeton & Morris, 1998) |
| `미세주입_배아_비율` | 미세주입 생성 배아 수 ÷ (총 생성 배아 수 + 1) | ICSI 활용 비율 (Palermo et al., 1992) |
| `불임_원인_개수` | 난관질환 + 남성요인 + 배란장애 + 자궁내막증 합산 | 복합 원인일수록 난이도 높음 |
| `본인_난자_여부` | 난자 출처 = 본인 제공 → 1 | 난자 출처별 성공률 차이 (Kawwass et al., 2013) |
| `기증_난자_여부` | 난자 출처 = 기증 제공 → 1 | 난자 출처별 성공률 차이 |
| `난자_기증자_사용여부` | 난자 기증자 나이 ≠ 알수없음 → 1 | Disguised Missing 처리 |
| `정자_기증자_사용여부` | 정자 기증자 나이 ≠ 알수없음 → 1 | Disguised Missing 처리 |
| `배아_이식_경과일_결측여부` | 결측이면 → 1 | 결측 자체가 "이식 안 함" 신호 |
